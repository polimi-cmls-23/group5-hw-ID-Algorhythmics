# YASC

This project is to to develop an interactive beatboxing recorder performance tool, incorporating
Joy-Con controllers, Node.js for data handling, and SuperCollider for sound synthesis.

## Requirements
+ node.js>=v16
+ SuperCollider software

## Installation

```bash
cd /html
npm install

cd /server
npm install
```

## Usage
1. Run server first.
```bash
cd /server
node index.js
```
2. Open the address shown in console.
3. Open the files in /supercollider in SuperCollider and run relative codes.
